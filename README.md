# Cybersecurity Home Labs

Labs práticos de cibersegurança montados em ambiente virtualizado (VirtualBox).
Objetivo: simular infraestrutura corporativa real para aprendizado de Blue Team e Red Team.

## Ambiente

- **Host:** Windows 10, 16GB RAM
- **Hypervisor:** VirtualBox 7.x
- **Rede:** pfSense como gateway/firewall central

## Labs

### Lab 1 — Firewall & Active Directory
Ambiente corporativo simulado com firewall perimetral, domínio Windows e monitoramento de endpoints.

**Tecnologias:** pfSense · Windows Server 2022 · Active Directory · Windows 10 · Sysmon
